
# NextJs-TypeScript BoilerPlate.  
This is the template i use for my web development based on Next.Js

## Featuring 
- [x] Nextjs
- [x] React
- [x] Context
- [x] Typescript
- [x] Webpack
- [x] Sass
- [x] Eslint

## Installation

Use the package manager [yarn](https://yarnpkg.com/en/docs/install) to install dependency.

```bash
yarn
```

## Running

### Development

```bash
yarn dev
```

### Production

```bash
yarn build
yarn start
```
